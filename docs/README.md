## 📌 **重要提醒**

### ⚠️ **使用前必读**

**在测试新功能前，必须完成：**

1. **执行数据库迁移**（5 分钟）
   - 参考：[Supabase 数据库迁移指南](./SUPABASE_MIGRATION_INSTRUCTIONS.md)
   - 创建 user_lists 和 user_list_items 表

2. **配置 Supabase 邮件服务**（2 分钟）
   - 参考：[Supabase 快速设置指南](./SUPABASE_QUICK_START.md)
   - 启用 Email Provider
   - 配置重定向 URL

3. **验证环境变量**（1 分钟）
   - 参考：[Vercel 配置完成报告](./VERCEL_CONFIG_COMPLETE.md)
   - 确认 Vercel 环境变量已配置
